## Overview

HoYoVista supports four HoYoverse games with varying feature availability. All games require linking your HoYoLAB account via `/hoyolink`.

## Genshin Impact

<Card title="Genshin Impact" icon="sword">
  **Game ID:** `2`
  
  **API Identifier:** `hk4e_global`
  
  **Act ID:** `e202102251931481`
</Card>

### Supported Features

| Feature | Status | Notes |
|---------|--------|-------|
| Daily Check-in | ✅ Full Support | Automatic and manual |
| Real-Time Notes | ✅ Full Support | Resin, commissions, realm currency, transformer |
| Profile View | ✅ Full Support | Uses Enka Network API |
| Income/Ledger | ✅ Full Support | Primogems and Mora tracking |
| Code Redemption | ⚠️ Maintenance | Feature temporarily disabled |

### Real-Time Notes Data

**Original Resin**
- Current resin count (max 200)
- Time until fully replenished
- Recovery timestamps

**Daily Commissions**
- Task completion status (4 commissions)
- Encounter Points status
- Long-Term Encounter Points accumulation
- Daily commission rewards claimed

**Enemies of Note**
- Weekly boss discount status
- Remaining resin cost-halving opportunities

**Jar of Riches**
- Current realm currency (Serenitea Pot)
- Maximum capacity based on trust rank
- Time until full
- Requires Serenitea Pot unlocked

**Parametric Transformer**
- Availability status
- Cooldown remaining in days
- Requires transformer obtained in-game

### Profile & Stats

- Adventure Rank (AR)
- Total achievements
- Characters with max friendship
- Spiral Abyss progress (floor-chamber-stars)
- Imaginarium Theater progress (act and stars)
- Stygian Onslaught difficulty and clear time
- Namecard display
- Profile signature

### Income Reports

- Monthly Primogem earnings breakdown
- Monthly Mora earnings
- Daily earnings (for current month)
- Comparison with previous month (percentage change)
- Sources: Daily Activities, Quests, Spiral Abyss, Events, Mail, etc.
- Primary source analysis
- Excludes top-up purchases

<Note>
Data has approximately 1-hour delay from in-game actions.
</Note>

## API Endpoints

HoYoVista uses the following HoYoLAB API endpoints:

| Endpoint | Purpose |
|----------|----------|
| `https://bbs-api-os.hoyolab.com` | Main API for game records and user data |
| `https://api-account-os.hoyolab.com` | Account authentication |
| `https://sg-hk4e-api.hoyolab.com` | Genshin Impact specific APIs |
| `https://sg-public-api.hoyolab.com` | Public game data |
| `https://sg-act-nap-api.hoyolab.com` | Zenless Zone Zero check-ins |
| `https://act.hoyolab.com` | Event and check-in actions |

## Technical Details

### Game Abbreviations

Used internally by the bot:
- Genshin Impact: `genshin`
- Honkai: Star Rail: `hkrpg`
- Honkai Impact 3rd: `honkai3rd`
- Zenless Zone Zero: `zzz`

### Event Paths

Used for check-in API calls:
- Genshin Impact: `sol`
- Honkai: Star Rail: `luna/hkrpg/os`
- Honkai Impact 3rd: `mani`
- Zenless Zone Zero: `luna/zzz/os`

### Region Codes

Supported server regions:
- North America: `os_usa`
- Europe: `os_euro`
- Asia: `os_asia`
- TW/HK/MO: `os_cht`

<Warning>
Chinese (CN) servers are not supported. Only global/overseas servers work with HoYoVista.
</Warning>
