HoYoVista provides comprehensive integration with HoYoLAB, bringing powerful features directly to Discord. Access your game data, claim daily rewards, and manage multiple accounts seamlessly.

## Core Features

All features support multiple HoYoverse games and can be used across Discord servers, DMs, and group chats.

<Note>
  Some features may have limited support for certain games. Check individual feature pages for game-specific availability.
</Note>

## Getting Started

Before using any features, you need to:

1. Register with `/register` to create your HoYoVista account
2. Link your HoYoLAB account using `/hoyolink`
3. Start using any feature command

<Info>
  All commands work in Discord servers, direct messages, and group chats. You can also use them in user-installed apps context.
</Info>

## Privacy & Security

HoYoVista takes your privacy seriously:

- Your HoYoLAB cookies are securely encrypted
- UID censoring is available for screenshots
- You control what data is collected for analytics
- All data is stored in accordance with our privacy policy

Learn more about privacy settings in the [Privacy Configuration](/configuration/privacy) section.

## Automation

Many features can be automated to save you time:

- **Auto Check-In**: Automatically claim daily rewards
- **Auto Redeem**: Automatically redeem new promo codes
- **Notifications**: Get notified about important events

Explore the [Automation](/automation/overview) section to set up automated tasks.
